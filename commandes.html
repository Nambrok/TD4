  • Commandes Git
git init nouveau_projet
Crée un dossier vide nouveau_projet, prêt pour le contrôle de versions.
git clone https://github.com/moi/mon_projet
Crée un dossier contenant une copie du projet et garde une référence vers le projet d’origine (remote origin) pour les push et les pull.
git diff
Montrer les différences par rapport à la dernière sauvegarde.
git show
Monter les changement effectuer par un commit
gitk
Explorateur graphique
git add fichier1.c fichier2.html
Ajouter les fichiers à sauvegarder.
git gui
Interface graphique
git log
La liste de tous les commits.
git show c797DHUCSUF8Q8F8F
Examiner un seul commit.
git status
Examiner le répertoire
git pull
Appliquer les changement même si il y a des conflits.
git push
Envoyer les changement à la branche tracée.
git merge master
importe les commits de la branche master avec la branche active
